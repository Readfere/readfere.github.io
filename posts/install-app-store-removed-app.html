安装 App Store 下架应用

2024-03-12T06:48:00.000Z2024-03-12T06:48:00.000Z

我一共有两台 iPhone 设备和一台 mac 设备，最近我其中一台 iPhone 出现了 Handoff 功能失效的情况（也就是剪贴板接力功能，从一台设备复制，另一台设备粘贴），这台 iPhone 设备先后使用了重新登录 iCloud 账号，重置网络设置，重置所有设置等等方法，到最后都没有任何效果，而且也不是网络的问题。

后来我咨询了苹果的售后，他们同样也让我进行以上的设置，我之前基本把他们说的解决方案都做过了，然后她祭出了最后一个大招，备份系统，还原所有设置。

最后我经历了两次还原设置后并恢复了系统备份文件，终于 Handoff 恢复了正常（有的时候也抽风，比如只能接力网址，文字就无法接力，我最后觉得是系统的 bug，很玄学）

最最让我焦虑的事来了，那就是我恢复系统之后发现，“shu 文件管理 ”这个 app 由于已经在 App Store 下架，下载不了了，这样导致的结果是我的 shu 文件夹里面保存的一千多个文件也回不来了，这下我着急了。我要做的就是找到能重新安装 shu 的方法，在网上找了很久经历了很多失败后，我终于找到了以下方案！

前置条件：

  1. 下载 IPAdown 软件（此为 Windows 软件）——功能是下载旧版 app
  2. Windows 上安装 iTunes 和 iCloud，两个都是最新版就行
  3. 下载 Sideloadly ——此为自签名 ipa 工具

下载并安装了以上软件之后：

  1. 首先打开 IPAdown 软件

    在 Google 搜索你想要下载的下架 app 的软件 id，然后输入到上图的软件 id 框中，点击“软 id 查旧版本”然后下面会出现软件的各版本 id，点击一个你想要下载的软件版本，然后点击右上角的：“下载”按钮，就会下载该 app 的旧版本 ipa。
  2. 得到这个 ipa 之后你需要用你的 Apple ID 登录你的 iTunes 和 iCloud
  3. 用数据线连接你的 iPhone 到电脑并打开 Sideloadly 软件

    根据以上截图的步骤操作 1. 选择连接的 iPhone 2. 点击“Advanced Options” 3. 在 signing mode 中选择 “normal install”（这步对于已下架的 app 安装很重要，我试过用 apple id sideload 没有成功）
  4. 将你的 ipa 文件拖入到 4️⃣ 的框中
  5. 点击“start”开始安装
  6. 不出意外就会成功将 ipa 文件安装到你的手机里面。
    注意：自签名软件需要每七天按照以上步骤重新签名一次
